# Retail Sales Analysis using SQL

## 📊 Project Overview
This project focuses on analyzing retail sales data using SQL to extract meaningful insights about sales performance, customer behavior, and product trends.

## 🛠 Tools Used
- SQL (MySQL)

## 📁 Dataset
Retail sales dataset containing transaction details such as date, category, customer, quantity, and total sales.

## 🔍 Key Analysis Performed
- Retrieved sales data for specific dates and conditions
- Analyzed category-wise sales and total orders
- Calculated average customer age for specific categories
- Identified high-value transactions
- Analyzed transactions by gender and category
- Found best-performing months using window functions
- Identified top 5 customers based on total sales
- Calculated unique customers per category
- Created time-based sales shifts (Morning, Afternoon, Evening)

## 📈 Key Insights
- Certain categories contribute significantly more to revenue
- Seasonal trends impact monthly sales performance
- A small group of customers contributes to high sales
- Sales activity varies across different times of the day
