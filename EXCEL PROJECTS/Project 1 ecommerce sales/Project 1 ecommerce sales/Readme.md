# Ecommerce Sales Analysis Dashboard (Excel)

## 📊 Project Overview
This project focuses on analyzing ecommerce sales data using Microsoft Excel. The goal was to transform raw data into meaningful insights through interactive dashboards and visualizations.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## 📁 Dataset
Ecommerce sales dataset containing information about:
- Order Date
- Product and Category
- Sales / Revenue
- Region / Country
- Profit

## 🔍 Key Analysis Performed
- Monthly revenue trend analysis
- Category and sub-category performance
- Top 10 products by revenue
- Region-wise revenue comparison
- Profit analysis by country
- Interactive filtering using slicers (Year, Category)

## 📈 Key Insights
- Sales peak in October and December
- Home & Kitchen category generates highest revenue
- USA and Mexico contribute highest profit
- Certain products consistently outperform others

## 📊 Dashboard Features
- Clean and interactive layout
- Dynamic filtering using slicers
- Multiple visualizations for better understanding
- Business-focused insights

## 📂 Files Included
- Excel Dashboard File (.xlsx)
- Dashboard Screenshot

## 🚀 Conclusion
This project helped in understanding how Excel can be used as a powerful tool for data analysis and visualization. It demonstrates the ability to convert raw data into actionable business insights.
